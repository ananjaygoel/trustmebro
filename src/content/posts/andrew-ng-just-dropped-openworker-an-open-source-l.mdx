## What's Happening

Here's the thing: Andrew Ng has dropped OpenWorker, an MIT-licensed desktop AI agent that returns finished deliverables instead of chat replies.

It runs a local Python agent server under a Tauri shell, supports 30 curated tool-calling models plus fully local Ollama, and gates every write, shell command and off-machine action behind a typed risk engine.
