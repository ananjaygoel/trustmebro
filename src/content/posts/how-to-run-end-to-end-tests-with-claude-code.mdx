## Why This Matters

End-to-end tests are basically tests where you make the coding agent open a browser and click through it and test its implementations completely. In that way, an alternative testing scheme could be unit tests or integration tests. End-to-end tests are solid because they test the app just like how a user would use the app, which ensures that you are able to properly test the app and discover issues that the user might experience.
