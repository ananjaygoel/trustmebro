## What's Happening

Real talk: LlamaIndex’s legal-kb is a public reference app that gives agents filesystem-style access to a document knowledge base on Index v2.

It exposes retrieve (hybrid semantic search), find, read, and grep as tools, with automatic per-file versioning and visual citations. The stack is TanStack Start, AI SDK 6 (ToolLoopAgent), Prisma, and WorkOS. (it feels like chaos)

## The Details

Editors Pick Agentic AI Technology AI Shorts AI Applications New Releases Open Source Tech News LlamaIndex ‘legal-kb’: Agentic Retrieval over Index v2 with retrieve, find, read, and grep Tools By Michal Sutter - LlamaIndex has published legal-kb , a public reference app on GitHub. It is described as a knowledge base for legal documents, powered v2 (the LlamaParse Platform).

The project demonstrates a pattern the team calls a Retrieval Harness for agentic retrieval. The approach differs from single-shot retrieval.

## Why This Matters

Instead of one embedding search per query, an agent is given filesystem-style tools. It can then crawl a large, evolving knowledge base to solve a task. The tools mirror operations engineers already know: semantic and keyword search, regex grep, file search, and read.

This adds to the ongoing AI race that's captivating the tech world.

## Key Takeaways

- Legal-kb is a working TanStack Start web app, not a library.
- You sign in, create a project, upload files, and chat with an agent.
- Each project is mirrored as a managed LlamaCloud Index v2.
- Uploaded files are parsed and indexed automatically in the background.
