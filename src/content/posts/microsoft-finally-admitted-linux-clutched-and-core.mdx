## Why This Matters

It includes more than 75 Unix-style command-line tools, including staples found on pretty much every Linux install, like ls, cp, mv, grep, find, and dozens more. All the commands run natively on Windows, with no need for WSL, a virtual machine, or a Bash workaround. The tools are compiled as native Windows binaries, and can be installed with a single WinGet command .

This is part of the broader shift happening across the tech industry right now.

## The Bottom Line

Microsoft framed the announcement as one that improves the developer experience switching for those who regularly bounce between Windows, Linux, and macOS. But it also acknowledges the impact that Linux had on the developer tooling space.
